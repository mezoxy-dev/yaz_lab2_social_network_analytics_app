# yaz_lab2_social_network_analytics_app
Proje, ilişkileri graf olarak modelleyen bir Sosyal Ağ Analizi Uygulaması geliştirmeyi amaçlar. Öğrenciler; ağı dinamik yöneten, en kısa yolları ve toplulukları görselleştiren, grafı renklendiren bir sistem tasarlayacaktır. Hedef; nesne yönelimli programlama, veri yapıları ve algoritma analizi becerilerini bütünleşik ölçmektir.
